Projects

Find-a-Pet React Site

This site mimics the styling and functionality of the Petfinder website. I used React to create dynamically-populated content cards to showcase each animal on the website. The site is fully responsive and easily scales to any amount of animals added to the database. It allows the user to filter through animals based on type and add favorites to a list of "faves" for future perusal.

GitHub User Map

I used this project to hone my skills with APIs. By querying four different APIs--from Google, GitHub and Geomaps--I used vanilla JavaScript to display an interactive map of cities with their corresponding numbers of GitHub users. With this project I also delved deeper into asynchronous functions, Node.js and Heroku’s config variables feature to protect my API keys.

Responsive Design

The webpage you're looking at is an example of my HTML and CSS skills. Without using any frameworks, it incorporates three different layouts that respond to desktop, tablet and mobile screens. In addition to engineering the functionality of web applications, I have a knack for writing excellent copy and an eye for design.
